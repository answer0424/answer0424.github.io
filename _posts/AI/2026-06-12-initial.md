## Context Window

- 모델이 한 번에 기억할 수 있는 입력 범위
- 예) 8k, 32k, 128k, 1M 토큰
- 의미:

```text
사용자 질문
+
이전 대화
+
첨부문서
+
시스템 프롬프트
```

- 예를 들어 128k 모델이면 소설 수백 페이지 수준까지 한 번에 처리 가능

## Skill / Tool / Plugin

#### Tool

- 모델이 외부 시스템을 호출하는 기능

```text
날씨 조회
DB 조회
API 호출
메일 발송
캘린더 등록
```

- LLM은 판단하고 Tool이 실제 작업 수행

<br>

#### Skill

- 대체로 업무 능력
- 예시
  - SQL Skill
  - Coding Skill
  - Customer Support Skill

<br>

#### MCP(Model Context Protocol)

##### 목적

- AI와 외부 시스템 연결 표준화

> 기존

```text
ChatGPT -> Jira API
ChatGPT -> GitHub API
ChatGPT -> Notion API
```

- 전부 따로 개발

🔽🔽🔽

> MCP 등장 이후

```text
AI
 │
MCP
 │
 ├─ Jira
 ├─ GitHub
 ├─ Notion
 ├─ Slack
 └─ DB
```

- 한번 규격 맞추면 여러 AI에서 사용 가능

## Agent

- 단순 채팅봇이 아니라 목표를 달성하는 AI

> 기존 LLM

```text
질문
↓
답변
```

> Agent

```text
목표
↓
계획
↓
도구 호출
↓
실행
↓
검증
↓
재시도
↓
완료
```

> 예시

```text
다음주 제주도 여행 계획 짜줘

- Agent

1. 항공권 검색
2. 호텔 검색
3. 날씨 조회
4. 일정 생성
5. 결과 정리
```
