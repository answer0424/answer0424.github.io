## Context Window

- 모델이 한 번에 기억할 수 있는 입력 범위
- 예) 8k, 32k, 128k, 1M 토큰
- 의미:

```text
사용자 질문
+
이전 대화
+
첨부문서
+
시스템 프롬프트
```

- 예를 들어 128k 모델이면 소설 수백 페이지 수준까지 한 번에 처리 가능

## Skill / Tool / Plugin

#### Tool

- 모델이 외부 시스템을 호출하는 기능

```text
날씨 조회
DB 조회
API 호출
메일 발송
캘린더 등록
```

- LLM은 판단하고 Tool이 실제 작업 수행

<br>

#### Skill

- 대체로 업무 능력
- 예시
  - SQL Skill
  - Coding Skill
  - Customer Support Skill
