## AWS Management Console

- 웹 기반 사용자 인터페이스로 AWS 서비스를 시각적으로 쉽게 액세스하고 관리할 수 있음
- 이동 중에도 AWS Management Console 어플리케이션을 통해 리소스를 빠르게 확인할 수 있음

<br>

## AWS Command Line Interface (AWS CLI)

- 통합 명령줄 도구로, 한 번 설치하고 구성하면 여러 AWS 서비스를 커맨드라인에서 제어할 수 있음
- 스크립트를 통해 자동화된 관리 작업을 만들고 반복 실행 가능

<br>

## AWS CloudShell

- 웹 브라우저에서 사전 인증된 셸 환경을 제공하며, AWS 자격 증명을 자동으로 사용함
- CloudShell은 콘솔을 종료하지 않고도 명령을 실행할 수 있도록 해줌

<br>

## Software Development Kits (SDKs)

- 다양한 프로그래밍 언어/플랫폼에 맞춘 SDK를 통해 API 방식으로 AWS 서비스를 어플리케이션에 통합할 수 있음
- 개발자가 어플리케이션 코드 안에서 AWS를 직접 호출하고 관리할 수 있도록 해줌

<br>

## 요약

| 접근 방식 | 사용 목적 |
|-|-|
| AWS Management Console | 시각적 인터페이스를 통한 서비스 관리 |
| AWS CLI | 명령 기반 관리 및 자동화 스크립트 실행 |
| AWS CloudShell | 브라우저 기반 명령줄 환경 |
| SDKs | 개발 코드에서 AWS API 호출 |
