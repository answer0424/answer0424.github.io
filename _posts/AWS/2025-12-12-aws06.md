## Amazon Athena

- 서버리스 인터랙티브 쿼리 서비스로, Amazon S3에 저장된 데이터를 표준 SQL로 빠르게 분석할 수 있음
- 인프라 관리가 필요 없고, 쿼리 실행량 만큼만 비용 지불함
- AWS Glue Data Catalog와 통합되어 메타데이터 관리를 지원

<hr>

## Amazon CloudSearch

- 웹사이트 또는 어플리케이션을 위한 검색 솔루션 서비스
- 34개 언어 지원 및 자동 완성, 하이라이트, 지리 공간 검색 등 기능 제공
- 관리형 서비스로 확장성, 운영 비용 절감 가능

<hr>

## Amazon DataZone

- 데이터 카탈로그 및 데이터 소비를 관리하고 제공하는 데이터 관리 서비스
- AWS, 온프레미스, SaaS 등 다양한 위치의 데이터를 통합적으로 다루고 보안 접근을 제공

<hr>

## Amazon EMR

- 클라우드 기반 빅 데이터 플랫폼으로, Apache Spark, Hive, HBase, Flink 등 오픈 소스 도구를 활용해 대규모 데이터 처리가 가능
- 클러스터 설정, 운영, 확장을 자동화하고 페타바이트 규모 분석 처리를 지원

<hr>

## Amazon FinSpace

- 금융 서비스 산업용 데이터 분석 플랫폼
- 대규모 금융 데이터의 검색, 준비, 분석을 몇 분 내에 수행하도록 설계되어 있음

<hr>

## Amazon Kinesis

- 실시간 데이터 수집, 처리, 분석을 지원
- 구성 요소
  - Amazon Kinesis Data Streams: 실시간 데이터 스트리밍
  - Amazon Managed Service for Apache Flink: Apache Flink 기반 실시간 데이터 처리
  - Amazon Kinesis Video Streams: 실시간 비디오 스트리밍
