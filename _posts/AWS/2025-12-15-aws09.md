## 개요

- AWS 클라우드 인프라 기반으로 확장성, 신뢰성, pay-as-you-go 요금 체계, 머신 러닝 등을 갖춘 비즈니스용 애플리케이션을 제공

<hr>

## 서비스 목록 및 설명 

#### AWS AppFabric

- SaaS 애플리케이션 전반에서 보안 데이터를 집계, 정규화하는 완전 관리형 서비스
- 여러 SaaS 애플리케이션을 코드 없이 연결해 관찰성, 생산성, 보안을 향상시키는 데 사용

<br>

#### Amazon Chime

- 온라인 회의, 화상 회의, 음성 통화, 채팅 기능을 제공하는 통합 커뮤니케이션 서비스
- 조직 내외부와 콘텐츠 기능을 지원

<br>

#### Amazon Chime SDK

- 개발자가 실시간 음성, 비디오, 메시징 기능을 애플리케이션에 쉽게 추가할 수 있도록 지원하는 SDK

<br>

#### Amazon Connect

- 클라우드 기반 옴니채널 고객센터 서비스
- 사용량 기반 요금, 그래픽 인터페이스를 통합 고객 응대 흐름 설계, 성능 지표 추적 등의 기능을 갖추고 있음

<br>

#### Amazon Pinpoint

- 다중 참여 채널을 통한 타겟 메시지 전송 서비스
- 모바일/웹 앱과 통합하여 사용 패턴을 분석하고 메시지 응답을 추적할 수 있음

<br>

#### Amazon SES (Simple Email Service)

- 개발자가 애플리케이션에서 이메일을 보내는 데 사용하는 확장 가능하고 비용 효율적인 이메일 서비스
- 트랜잭션, 마케팅, 대량 메일 통신 등을 지원
