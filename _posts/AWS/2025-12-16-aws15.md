## AWS Developer Tools

> AWS 개발자 도구는 소프트웨어 개발 워크플로우 전반 코드 작성부터 빌드, 테스트, 배포, 디버깅, 최적화까지 지원하는 서비스 및 도구 모음
>
> 이를 통해 애플리케이션 개발 생산성을 높이고 DevOps 관행을 강화할 수 있음

<hr>

## 주요 개발자 도구 서비스 목록

#### AWS 인프라 컴포저 (Infrastructure Composer)

- 서버리스 애플리케이션 아키텍처를 시각적으로 구성할 수 있는 도구
- 브라우저 기반의 캔버스를 통해 리소스를 끌어다 놓고 구성하며, CloudFormation/SAM 아티팩트 생성을 지원

<br>

#### AWS Cloud9

- 브라우저 기반 클라우드 IDE
- 사전 설치된 편집기, 디버거, 터미널을 제공하여 로컬 개발 환경 설정 없이도 개발이 가능
- 서버리스 애츨리케이션 개발과 디버깅을 지원하며, 팀 협업도 가능

<br>

#### AWS CloudShell

- 브라우저에서 사전 인증된 셸 환경을 제공
- AWS CLI, SDK 도구를 설치/구성할 필요없이 터미널을 통해 AWS 리소스를 바로 제어할 수 있음

<br>

#### AWS CodeArtifact

- 완전관리형 소프트웨어 패키지 리포지토리 서비스
- 다양한 패키지 유형을 저장, 공유, 배포할 수 있으며, 공용 저장소에서 자동으로 패키지를 가져오도록 설정할 수 있음

<br>

#### AWS CodeBuild

- 완전관리형 빌드 서비스
- 소스 코드를 컴파일하고, 테스트를 실행하며 배포가능한 빌드 아티팩트를 생성
- 빌드 서버 관리가 필요없고 자동으로 확장됨
