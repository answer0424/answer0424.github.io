## AWS 최종 사용자 컴퓨팅

> AWS 최종 사용자 컴퓨팅 카테고리는 사용자에게 안전하고 확장 가능한 원격 데스트크 톱 및 애플리케이션 액세스 환경을 제공하는 서비스들을 포함
>
> 이를 통해 기업은 온프레미스 VDI의 복잡성을 줄이고, 클라루드 기반 EUC로 유연하게 마이그레이션할 수 있도록 지원

<hr>

## AWS EUC 서비스 목록 & 설명 

#### Amazon AppStream 2.0

- 완전관리형 애플리케이션 스트리밍 서비스
- 애플리케이션을 중앙에서 관리하고 사용자 디바이스에 설치 없이 스트리밍 방식으로 제공
- 글로벌 확장성, 보안 아키텍처 이용, GPU 집약적 워크로드도 지원

<br>

#### Amazon WorkSpaces

- 완전관리형 클라우드 데스크톱 서비스
- 몇 분 만에 Windows 또는 Linux 데스크톱을 프로비저닝하고 확장할 수 있음
- 하드웨어 인벤토리, OS 관리, VDI 인프라 관리 부담을 제거하면서 기존 온프레미스 VDI 대비 비용을 절감할 수 있음
- 지원 장치에서 어디서든 액세스 가능하여 원격 근무 환경에 적합

<br>

#### Amazon WorkSpaces Core

- 클라우드 기반 VDI를 제공
- 기존의 VDI 관리 소프트웨어와 연동하여 AWS 인프라 기반으로 확장할 수 있음
- 글로벌 가상 데스크톱을 제공하면서 높은 가용성 및 온디맨드 확장이 가능

<br>

#### Amazon WorkSpaces Thin Client

- AWS EUC 솔루션과 함께 작동하는 씬 클라이언트 디바이스
- 로컬 데이터 저장이나 승인되지 않은 애플리케이션 설치가 불가능하게 설계되어 엔드포인트 보안을 강화
- USB, 듀얼 모니터 등 주변기기 지원을 통해 실제 데스크톱과 유사한 환경을 제공

<br>

#### Aamzon WorkSpaces Web

- 브라우저 기반으로 내부 웹 사이트 및 SaaS 애플리케이션 등에 안전하게 액세스할 수 있는 서비스
- 기존 클라이언트 소프트웨어나 VPN이 필요하지 않아 간편하고 저비용으로 웹 중심 EUC 환경에 제공
