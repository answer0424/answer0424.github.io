## 개요

> AWS는 ML/AI 도입의 모든 단계에서 지원하는 폭넓은 ML/AI 서비스 세트를 제공
>
> 이에는 사전 학습된 AI 서비스, 관리형 ML 플랫폼, 대화형/생성형 AI, 그리고 도메인 특화 ML 서비스가 포함

<hr>

## AWS Machine Learning / AI 서비스 목록

#### Amazon Bedrock

- 파운데이션 모델을 API로 제공하는 완전 관리형 생성형 AI 서비스
- 다양한 모델과 Amazon Nova를 선택할 수 있으며, 사용자 데이터로 맞춤화 및 애플리케이션 통합이 가능

<br>

#### Amazon Augmented AI (A2I)

- 사람 검토가 필요한 워크플로우를 쉽게 구축할 수 있는 ML 서비스
- 자동화된 사람 검토 결합으로 ML 결과를 검증/보완할 수 있게 함

<br>

#### Amazon Comprehend

- 비정형 텍스트에서 언어, 키프레이즈, 감정, 엔터티 등을 추출하는 NLP 서비스
- AutoML을 통해 맞춤형 엔터티/분류 모델을 생성할 수 있음

<br>

#### Amazon Forecast

- 시계열 예측을 위한 완전관리형 ML 서비스
- 과거 데이터 기반으로 수요, 리소스, 성과 추정 등 예측 모델을 자동 생성

<br>

#### Amazon Fraud Detector

- ML과 Amazon의 사기 탐지 전문 지식을 활용하여 사기 탐지 모델을 자동 구축 배포함
- 각 고객 데이터에 맞춘 맞춤형 예측이 가능

<br>

#### Amazon Lookout for Equipment

- 장비 센서 데이터를 분석해 이상/고장 전조 신호를 탐지하는 ML 서비스
- ML 전문 지식 없이 실시간 이상 감지가 가능

<br>

#### Amazon Lookout for Vision

- 컴퓨터 비전으로  결합/이상 탐지하는 서비스
- 제조 품질 검사/불량 탐지에 활용

<br>

#### Amazon Lex

- 대화형 인터페이스 구축을 위한 서비스
- 음성/텍스트 기반 봇이나 음성 응답 시스템을 쉽게 만들 수 있음

<br>

#### Amazon Polly

- 텍스트 ➡️ 음성 변환 서비스로, 자연스러운 음성 출력이 가능

<br>

#### Amazon Personalize

- 맞춤형 추천 시스템을 쉽게 구축할 수 있는 ML 서비스

<br>

#### AWS Panorama

- 엣지 컴퓨터 비전 플랫폼/SDK로 IP 카메라에 ML 기반 시각 분석 기능을 제공

<br>

#### Amazon Transcribe

- 자동 음성 인식(ASR) 서비스로 음성을 텍스트로 변환
- 의료 및 콜센터 특화 옵션을 제공

<br>

#### Amazon TextTract

- 스캔된 문서에서 텍스트/데이터 추출을 자동화하는 ML 기반 OCR 서비스

<hr>

## SageMaker AI 플랫폼 및 도구

> AWS는 또한 Amazon SageMaker 기반의 종합 ML 개발 플랫폼을 제공

<br>

#### Amazon SageMaker AI Autopilot

- 자동으로 테이블 데이터 기반 ML 모델 생성,훈련,조정을 수행
