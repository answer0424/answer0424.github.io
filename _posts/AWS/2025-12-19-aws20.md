## 개요

> AWS의 관리 및 거버넌스 서비스는 클라우드 환경을 더 빠르게 혁신하면서도 제어, 비용 효율, 규정 준수를 유지할 수 있도록 돕는 서비스

<hr>

## AWS 관리 및 거버넌스 서비스 목록

#### AWS Auto Scaling

- 어플리케이션 트래픽/리소스 사용량을 모니터링하고, 자동으로 컴퓨팅/데이터베이스/스케일링을 조정함
- EC2, ECS 작업, DynamoDB, Aurora 등 다양한 서비스에 대한 확장 계획을 생성하고 실행할 수 있음

<br>

#### AWS CloudFormation

- AWS 리소스 모음을 템플릿 기반으로 정의/배포/업데이트할 수 있는 서비스
- 인프라를 코드처럼 관리하고 버전/의존성을 제어할 수 있음

<br>

#### AWS CloudTrail

- AWS 계정 내 API 호출 및 활동 로그를 기록하는 서비스
- 호출자 ID, 요청 파라미터, 응답 요소 등을 기록하는 보안 분석, 감사 및 추적에 활용할 수 있음

<br>

#### Amazon CloudWatch

- AWS 및 온프레미스 자원/애플리케이션의 지표/로그/이벤트를 수집, 시각화, 분석을 함
- 알람 설정/자동화된 작업/모니터링을 통해 운영 상태를 통합적으로 파악할 수 있음

<br>

#### AWS Compute Optimizer

- 머신 러닝을 활용해 과거 사용률을 분석하고, 비용 및 성능 최적화 권장을 제공
- EC2, EBS, Lambda 등 자원에 대한 최적 구성을 추천함

<br>

#### AWS Console Mobile Application

- 이동 중에도 AWS 리소스를 모니터링 및 일부 관리할 수 있는 모바일 앱
- CloudWatch, Health Dashboard,  비용/상태 뷰 등을 통합 대시보드로 제공

<br>

#### AWS Control Tower

- 보안/규정 준수 기반의 멀티계정 환경 설정을 자동화함
- AWS Organizations, Service Catalog, Config 등과 연동해 중앙 거버넌스 체계를 구축함

<br>

#### AWS Config

- AWS 자원 구성 히스토리, 상태, 변경사항을 추적/기록하는 서비스
- Config Rules를 통해 리소스 구성 규칙 준수 여부 자동 검사가 가능

<br>

#### AWS Health

- AWS 서비스 상태 변화를 사용자 맞춤형으로 보여주는 대시보드
- 예정된 유지보수나 이벤트, 장애 영향 정보를 알림/가이던스로 제공

<br>

#### AWS Launch Wizard

- SAP, SQL server 등 애플리케이션 전용 배포 리소스를 자동 프로비저닝하는 가이드형 서비스
- 설정 입력만으로 필요한 AWS 리소스를 식별/구성/배포할 수 있음

<br>

#### AWS License Manager

- 라이선스를 AWS 및 온프레미스에 걸쳐 중앙으로 관리할 수 있게 해주는 서비스
- 라이선스 규칙을 정의하여 초과 사용을 방지할 수 있음

<br>

#### Amazon managed Grafana

- Grafana 기반의 완전 관리형 시각화/모니터링 대시보드 서비스 
- CloudWatch, OpenSearch, Prometheus 등 다양한 데이터 소스와 통합됨

<br>

#### Amazon Managed Service for Prometheus

- 서버리스 Prometheus 호환 모니터링 서비스로, 컨테이너 워크로드 지표 수집/저장/쿼리를 자동 확장함
