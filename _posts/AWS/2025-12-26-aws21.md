## 개요

> AWS는 디지털 콘텐츠 생성/변환/전송 및 배포를 클라우드에서 쉽고 유연하게 할 수 있도록 설계된 미디어 서비스, 소프트웨어, 어플라이언스 세트를 제공

<hr>

## AWS 미디어 서비스 목록 및 설명

#### AWS Elastic Transcoder

- 클라우드 기반 미디어 트랜스코딩 서비스
- 소스 형식의 비디오/오디오 파일을 스마트폰, 태블릿, PC 등 다양한 디바이스에서 재생 가능한 형식으로 변환할 수 있게 해줌
- 확장성과 비용 효율성이 좋음

<br>

#### AWS Interactive Video Service (IVS)

- 완전관리형 라이브 스트리밍 솔루션
- 빠르게 설정할 수 있고 대화형 비디오 경험 구축에 적합
- AWS IVS Player SDK 및 메타데이터 API를 통해 커스터마이징도 가능

<br>

#### Amazon Nimble Studio

- 클라우드 기반 콘텐츠 제작 스튜디오 서비스
- 아티스트가 가상 워크스테이션, 고속 스토리지 및 확장 렌더링 리소스를 이용해 영상, 애니메이션 등 콘텐츠를 제작할 수 있게 함

<br>

#### AWS Elemental Appliances and Software

- 온프레미스 또는 데이터센터에서 비디오 처리 및 전달 기능을 제공하는 물리적 장비/소프트웨어 솔루션
- AWS Elemental Live, Server, Conductor 등 다양한 구성 옵션을 제공하며, AWS 클라우드 기반 미디어 워크플로와 연결해 사용할 수 있음

<hr>

## AWS Elemental 미디어 서비스

#### AWS Elemental MediaConnect

- 고품질 라이브 비디오 전송 서비스
- 위성/광섬유 기반 전송의 비용/복잡성을 줄이면서 IP 기반 안정적 스트림 전송을 제공

<br>

#### AWS Elemental MediaConvert

- 파일 기반 비드오 트랜스코딩 서비스
- VOD 콘텐츠 처리를 위해 브로드캐스트급 기능을 제공
