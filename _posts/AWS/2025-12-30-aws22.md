## AWS Migration & Transfer 서비스 개요

> AWS는 애플리케이션 및 데이터의 평가, 이동, 현대화 전 과정을 지원하는 다양한 서비스와 도구를 제공
>
> 이를 통해 온프레미스 또는 다른 클라우드 환경에서 AWS로의 이전을 보다 쉽고 안전하게 할 수 있음

<br>

## 주요 서비스

#### AWS Application Discovery Service

- 온프레미스 데이터센터의 구성, 사용량, 동작 데이터를 수집하여 마이그레이션 계획 수립을 지원
- 수집된 데이터는 CSV로 내보내 분석 및 총 소유 비용 평가에 사용할 수 있으며, Migration Hub에서도 활용할 수 있음

<br>

#### AWS application Migration Service (AWS MGN)

- 소스 서버의 애플리케이션을 AWS로 중단 최소화 방식으로 이전할 떄 사용됨
- 서버 변환 자동화 및 마이그레이션 테스트 기능을 제공하여 안정적인 마이그레이션을 도움

<br>

#### AWS Database Migration Service (AWS DMS)

- 데이터베이스를 AWS로 쉽고 안전하게 마이그레이션 
- 소스 DB는 마이그레이션 중에도 계속 운영되며, 동종 및 이종 마이그레이션을 모두 지원함
- Amazon Redshift 같은 데이터 웨어하우스로 스트리밍 방식으로 데이터 통합/분석도 가능
- AWS DMS Serverless는 복제 인스턴스를 별도 프로비저닝하지 않아도 되는 유연성을 제공

<br>

#### AWS Mainframe Modernization Service

- 온프레미스 메인프레임 워크로드를 AWS 상의 관리형 런타임 환경으로 이전 및 현대화할 수 있게 해줌
- 하드웨어 및 인력 비용 부담 제거, 도구를 통한 필터링 및 변환, 현대화된 환경에서의 운영을 지원함

<br>

#### AWS Migration Hub

- 여러 AWS 및 파트너 마이그레이션 도구를 사용한 마이그레인션 전체 진행 상황을 단일 위치에서 추적할 수 있는 중앙 대시보드를 제공'
- 도구/서비스 종류와 상관없이 마이그레이션 진행 상태와 핵심 지표를 확인할 수 있어 관리 및 문제 식별을 용이하게 함

<br>


#### AWS Snow Family

- AWS Snowball, Snowball Edge 같은 물리적 디바이스를 이용해 네트워크 연결리 불안정하거나 대용량 데이터를 전송해야 하는 환경에서도 데이터를 AWS로 오프라인/온라인 이동할 수 있도록 함
- 일부 디바이스는 컴퓨팅 기능을 내장하여 엣지 환경 처리도 가능

<br>

#### AWS DataSync

- 온프레미스 스토리지와 Amazon S3 또는 Amazon EFS 등 AWS 스토리지 간 데이터 이동 자동화 서비스
- 스크립트 관리, 암호화 처리, 네트워크 최적화, 데이터 무결성 검증 등을 자동화해 데이터 전송을 간편화/고속화함

<br>

#### AWS Transfer Family

- SFTP, FTPS, FTP 등 기존 파일 전송 워크플로우를 AWS 스토리지로 그대로 연계할 수 있는 완전관리형 서비스
- 인증 시스템 통합, DNS 라우팅 등을 지원하여 기존 방식 변경 없이 AWS로의 차일 전송을 수행할 수 있음
