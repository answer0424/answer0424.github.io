## AWS 네트워킹 서비스

> AWS 클라우드에서 높은 신뢰성, 보안, 성능을 제공하는 폭넓은 네트워킹 및 콘텐츠 전송 서비스를 제공함 
>
> 이것이 사용자의 애플리케이션, 데이터 및 서비스가 최적의 성능과 연결성을 갖도록 돕는 핵심 인프라 컴포넌트

<hr>

## 네트워킹 & 콘텐츠 전송 주요 서비스

#### Amazon API Gateway

- API를 쉽게 생성/게시/관리/보호할 수 있는 완전관리형 API 서비스
- 백엔드 서비스와의 트래픽 관리, 권한 제어, 모니터링을 처리함
