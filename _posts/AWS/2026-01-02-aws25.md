## Amazon EC2 주요 기능

> Amazon EC2는 AWS 클라우드에서 온디맨드 방식으로 컴퓨팅 용량을 제공하는 서비스

<hr>

## 인스턴스 (Instances)

- 가상 서버 역할을 하는 EC2의 기본 실행 단위
- 인스턴스를 시작/중지/종료하는 등 직접 제어 가능

<hr>

## Amazon Machine Images (AMIs)

- 운영 체제와 필요한 소프트웨어 구성이 포함된 EC2 인스턴스 템플릿임
- 이 AMI를 기반으로 여러 인스턴스를 쉽게 생성할 수 있음
