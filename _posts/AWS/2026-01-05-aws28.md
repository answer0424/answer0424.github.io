## 개요

- AWS Management Console을 사용해 EC2 인스턴스 시작
- 인스턴스에 접속
- 인스턴스 정리

<hr>

## 핵심 구성 요소

| 구성 요소 | 설명 |
|-|-|
| 이미지 (AMI) | 운영 체제 포함 템플릿, 인스턴스 시작 기반 |
| 키 페어 | SSH/RDP 연결 시 본인 인증에 사용하는 공개/프라이빗 키 쌍 |
| 네트워크 | AWS 계정 전용 가상 네트워크, 기본 VPC가 자동 제공됨 |
| 보안 그룹 | 인스턴스 트래픽 제어를 위한 가상 방화벽 역할 |
| EBS 볼륨 | 루트 디스크 등 영구 저장용 블록 스토리지 |
