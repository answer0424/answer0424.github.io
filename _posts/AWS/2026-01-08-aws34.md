## 유료 AMI (Paid AMI)란?

- 운영 체제뿐 아니라 서드파티 소프트웨어 또는 상요 소프트웨어가 포함된 이미지
- 사용하려면 AMI 소유자가 제공한 요금/라이선스 계약에 동의한 후 구독해야 함
- 기본 AMI처럼 EC2 인스턴스로 시작할 수 있지만, 해당 라이선스 비용이 별도로 부과될 수 있음

<hr>

## 유료 AMI 찾는 방법

#### 1. AWS Management Console에서 찾기

1. Amazon EC2 콘솔에 로그인
2. 왼쪽 네비게이션에서 AMI 메뉴로 이동함
3. 퍼블릭 이미지 필터를 선택
4. 다음중 하나로 원하는 AMI를 찾음
   - 제품 코드를 알고 있는 경우 
     - 필터에서 제품 코드 = 값으로 입력하여 검색
   - 모르는 경우
     - Owner alias: `aws-marketplace` 필터를 선택해 AWS Marketplace AMI만 나열
5. 찾은 AMI ID를 저장

<br>

#### 2. AWS CLI로 찾기

```bash
aws ec2 describe-images --owners aws-marketplace
```

- 이는 `aws-marketplace` 소유자에 속한 AMI를 나열
- 필요하다면 추가 필터를 적용할 수 있음
- 예를 들어 특정 AMI ID의 제품 코드를 확인하려면

```bash
aws ec2 describe-images \
    --image-ids ami-0abcdef1234567890 \
    --query Images[*].ProductCodes[].ProductCodeId
```
