## AMI 도구 설치 - RPM 방식

> AMI 도구를 RPM 패키지로 설치하려면 다음 절차를 따름

#### 1. Ruby 설치

```bash
sudo yum install -y ruby
```

<br>

#### 2. AMI 도구 다운로드

```bash
wget https://s3.amazonaws.com/ec2-downloads/ec2-ami-tools.noarch.rpm
```

<br>

#### 3. RPM 서명 검증

```bash
rpm -K ec2-ami-tools.noarch.rpm
```

<br>

#### 4. RPM 설치

```bash
sudo yum install ec2-ami-tools.noarch.rpm
```
