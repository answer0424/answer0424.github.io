## 관련 API

| API | 역할 |
|-|-|
| CreateStoreImageTask | AMI와 해당 EBS 스냅샷을 S3 객체로 저장 |
| DescribeStoreImageTasks | 저장 작업의 진행 상태 확인 |
| CreateRestoreImageTask | S3 객체에서 AMI를 복원 |
