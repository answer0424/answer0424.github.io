## 기본 동작 원리

- EC2는 AMI가 인스턴스를 시작하는 데 마지막으로 사용된 날짜/시간 정보를 자동으로 보관함
- 이 값은 AMI 소유자에게만 보이며, 해당 AMI를 다른 계정이 소유하지 않으면 볼 수 없음
- AMI 사용 시점은 사용 후 약 24시간 뒤부터 표시됨
- 사용 데이터는 2017년 4월 이후 기록된 AMI에 대해서만 표시됨

---

## 콘솔에서 확인하는 방법

1. AWS Management Console에서 Amazon EC2 콘솔을 엶
2. 왼쪽 탐색에서 AMI를 선택
3. 필터에서 내 소유를 선택해 소유한 AMI만 표시
4. 확인하고 싶은 AMI를 선택
5. AMI 세부 정보 탭에서 마지막 시작 시간을 확인

---

## CLI에서 확인하는 방법

#### AMI 설명으로 확인

```bash
aws ec2 describe-images \
    --image-id ami-0abcdef1234567890 \
    --query Images[].LastLaunchedTime \
    --output text
```

- 출력 예:

```yaml
2025-02-17T20:22:19Z
```

--- 

## AMI 속성(describe-image-attribute)으로 확인

```bash
aws ec2 describe-image-attribute \
    --image-id ami-0abcdef1234567890 \
    --attribute lastLaunchedTime \
    --query LastLaunchedTime.Value \
    --output text
```
