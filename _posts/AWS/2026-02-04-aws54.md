## EC2 인스턴스 유형 파인더

> EC2 인스턴스 유형 파인더는 사용자의 요구 조건을 입력하면, 해당 워크로드에 적합한 인스턴스 유형을 추천 및 안내해주는 도구임

- 워크로드 유형
- 사용 사례
- 가격 대비 성능 우선순위
- CPU 제조업체 선호 
- 기타 세부 필터 조건

--- 

## 사용 시기

- 새 워크로드를 위한 인스턴스
- 가격 대비 성능을 중점으로 선정할 때
- ARM/인텔/AMD 중 APU 선호 기준이 있을 때

---

## 사용 방법

#### 1. 인스턴스 생성 흐름에서 접근

- 인스턴스 시작 과정 - 인스턴스 유형 부분의 조언 받기 선택
- 시작 템플릿 생성 - 인스턴스 유형 옆 조언 받기 클릭
- 인스턴스 유형 메뉴 - 인스턴스 유형 찾기 버튼 선택

<br>

#### 2. 요구 조건 입력

1. 워크로드 유형 선택
2. 사용 사례 지정
3. 가격 및 성능 우선 순위 선택
4. CPU  제조업체 선호 선택
