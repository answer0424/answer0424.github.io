## 권장 사항이란?

> Compute Optimizer는 CloudWatch 지표와 인스턴스 사양을 분석하여, 현재 워크로드에 대해 다음과 같은 인스턴스 상태를 평가하고 권장 타입을 제공


| 분류 | 의미 |
|---|---|
| Under-provisioned(과소 프로비저닝) | 워크로드의 성능 요구를 충족하지 못함 |
| Over-provisioned(과대 프로비저닝) | 성능 요구는 충족되지만 불필요한 자원이 존재함 |
| Optimized(최적화됨) | 현재 인스턴스가 적합하게 프로비저닝 되어 있음 |
| None(없음) | 권장 사항 생성 불가 |
