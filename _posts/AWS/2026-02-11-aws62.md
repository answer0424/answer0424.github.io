## 핵심 요약

- AWS EC2의 GPU 기반 인스턴스에서 NVIDIA GRID Virtual Applications 기능을 활성화하는 방법을 정리
- 활성화는 운영체제별 설정으로 이루어지며 인스턴스 생성 후 구성해야 함

--- 

## 개념

- NVIDIA GRID Virtual Applications는 NVIDIA GPU 기반 EC2 인스턴스에서 **가상 애플리케이션 실행(가속 GPU 사용)**을 허용하는 기능
- 일부 GPU 인스턴스에서는 Virtual Workstation 기능은 기본 활성화되어 있지만, Virtual Applications는 추가 구성 필요

---

## 사전 조건

- GPU 기반 EC2 인스턴스 실행 중이어야 함
- 운영체제는 Linux 또는 Windows여야 함
- NVIDIA 드라이버가 설치되어 있어야 GRID 설정 파일/레지스트리 옵션을 구성할 수 있음

---

## Linux 인스턴스에서 활성화

#### 1. 템플릿 파일을 복사하여 GRID 설정 파일 생성

```bash
sudo cp /etc/nvidia/gridd.conf.template /etc/nvidia/gridd.conf
```

<br>

#### 2. `gridd.conf` 파일 편집

- `FeatureType = 0`로 설정
- `IgnoreSP=TRUE`를 추가

```ini
FeatureType=0
IgnoreSP=TRUE
```
