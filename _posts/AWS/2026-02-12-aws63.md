## EC2 콘솔에서 인스턴스 페이지 열기

- AWS MAnagement Console에 로그인
- 상단 검색창에 EC2 입력 후 선택
- 왼쪽 탐색 창에서 인스턴스(Instances) 클릭
  - 방금 인스턴스를 만들고 성공 화면에 있다면 
    - 상단 브레드크럼에서 Instances 링크 선택하면 바로 이동 가능

---

## 생성된 인스턴스 찾기

- 이름 열에서 자신이 입력한 인스턴스 이름으로 찾음
  - 이 값은 인스턴스 생성 시 입력한 태그의 Name 값에 해당
- 목록이 많은 경우 검색/필터링 기능으로 이름 또는 상태로 좁힐 수 있음

---

## 인스턴스 상세 확인

#### 1. 인스턴스 목록에서 해당 인스턴스의 Instance ID 열 값을 확인

- 형식: `i-xxxxxxxxxxxxxxxxx`

<br>

#### 2. Instance ID 링크 클릭

- 해당 인스턴스의 상세 정보 페이지가 열림

#### 3. 상세 정보에서 다음을 확인/검토

- 인스턴스 상태
- 인스턴스 유형
- 퍼블릭/프라이빗 IP
- 보안 그룹, 서브넷, AMI 정보
- 태그 및 모니터링 상태

---

## 팁: 필드 빠르게 찾기

- 상세 정보가 많을 경우
  - 브라우저에서 Ctrl+F / Cmd+F로 원하는 필드 이름 검색
  - 예: `Instance type`, `Public IPv4 address` 등
