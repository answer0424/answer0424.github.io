## 핵심 요약

- Launch Instance Wizard는 EC2 인스턴스를 시작할 때 사용하는 웹 기반 구성 마법사
- 대부분 파라미터에 대해 기본값을 제공하며, 이를 그대로 사용하거나 필요에 따라 수정할 수 있음
- 키 페어만 필수로 지정해야 하며, 나머지는 기본값 그대로 빠르게 시작 가능
- 인스턴스 실행 중인 동안에는 요금이 발생함

---

## 인스턴스 시작 흐름

#### 1. 콘솔 진입

1. AWS Management Console에 로그인
2. 상단 검색 또는 서비스 목록에서 EC2 선택
3. EC2 대시보드에서 인스턴스 시작 클릭

<br>

#### 2. 이름 및 태그 지정

- Name 필드에 인스턴스 식별용 이름 입력
- 이 값은 Name 태그로 인스턴스에 자동 생성됨

<br>

#### 3. AMI 선택

- Amazon Machine Image: OS + 기본 소프트웨어 템플릿
- 퀵 스타트 목록에서 운영체제 선택
- 프리티어 퍼블릭을 선택 가능

<br>

#### 4. 키페어 선택

- SSH/RDP 접속용 키 페어 지정
  - 기존 키 선택 또는 새 키 생성
  - 새 키 생성 시 .pem 파일 다운로드 저장 필수 
- 키 페어는 마법사의 유일한 필수 파라미터

<br>
