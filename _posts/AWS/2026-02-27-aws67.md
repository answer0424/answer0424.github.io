## 핵심 요약

- 시작 템플릿은 EC2 인스턴스 시작 파라미터를 저장해 두는 재사용 가능한 구성 리소스
- 템플릿을 지정해 인스턴스를 시작하면, 매법 파라미터를 입력할 필요 없이 템플릿에 저장된 설정값을 활용하여 빠르게 배포할 수 있음
- 템플릿에는 버전 관리가 가능하며, 버전을 선택해 인스턴스를 시작할 수 있음
- 시작 템플릿은 EC2 콘솔 외에도 Auto Scaling, EC2 Fleet, Spot Fleet 등에서도 사용됨

---

## 개념: 시작 템플릿이란

- EC2 인스턴스를 시작할 때 필요한 파라미터 묶음을 저장해 두는 리소스
- 저장된 파라미터 예:
  - AMI ID
  - 인스턴스 유형
  - 키 페어
  - 네트워크 / 서브넷
  - 보안 그룹
  - 태그 등
- 템플릿을 사용하면 반복적인 인스턴스 시작 작업을 표준화/자동화할 수 있음

---

## 시작 템플릿 버전

- 각 템플릿은 여러 버전을 가질 수 있음
- 버전 별로 서로 다른 시작 파라미터를 정의 가능
- 인스턴스를 시작할 때 특정 버전을 선택하거나 기본 버전을 사용할 수 있음
