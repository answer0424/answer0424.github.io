## 핵심 요약

- Launch More Like This는 기존 EC2 인스턴스의 구성 정보를 기준 템플릿처럼 활용하여 빠르게 새 인스턴스를 시작하는 기능
- 기존 인스턴스를 참조 원본으로 하여 AMI, 인스턴스 유형, 네크워크, 스토리지, 태그 등을 자동으로 불러오고 수정해 시작할 수 있음
- 템플릿 생성 없이도 일회성으로 유사 구성 인스턴스 빠르게 생성 가능
