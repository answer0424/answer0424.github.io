## 핵심 요약

- Launch More Like This는 기존 EC2 인스턴스의 구성 정보를 기준 템플릿처럼 활용하여 빠르게 새 인스턴스를 시작하는 기능
- 기존 인스턴스를 참조 원본으로 하여 AMI, 인스턴스 유형, 네크워크, 스토리지, 태그 등을 자동으로 불러오고 수정해 시작할 수 있음
- 템플릿 생성 없이도 일회성으로 유사 구성 인스턴스 빠르게 생성 가능

---

## 기능 개요

- AWS 콘솔 EC2 인스턴스 목록에서
  - 기존 인스턴스 선택
  - Actions ➡️ Launch More Like This 선택
  - 기존 구성값을 기반으로 새 인스턴스 설정화면이 열림
- 기존 인스턴스 구성 자료를 자동으로 불러와 재사용/수정/적용함

---

## 작동 원리

- 기존 인스턴스 정보 수집
  - EC2 콘솔이 선택한 인스턴스의 시작 파라미터를 기반으로 설정값을 추출
- Launch More Like This 화면 오픈
  - 해당 구성값을 인스턴스 
- 파라미터 확인/수정 후 시작
  - 필요 시 네트워크, 보안 그룹, 스토리지, 태그 같은 값을 수정한 뒤 인스턴스를 생성

---

## 포함/불포함 구성

#### 자동 불러오는 항목

- AMI ID
- 인스턴스 유형
- 키 페어 이름
- 네트워크/서브넷
- 보안 그룹
- 스토리지 볼륨 설정
- 태그 값
  - 기존 인스턴스 기준이며, 일부 옵션에 대해서는 수동 수정이 필요할 수 있음

<br>

#### 자동 채워지지만 검토가 필요한 항목

- 퍼블릭 IP 자동 할당 여부
- VPC/서브넷 간 변경
- 보안 그룹에서 열린 포트
- IAM 역할
  - 기존 인스턴스와 동일한 네트워크/권한 설정이 항상 유효한 것은 아니므로 검토 권장

---

## 사용 시나리오

| 상황 | 활용 예 |
|-|-|
| 동일 구성 인스턴스 여러 대 생성 | LMLT로 빠르게 시작 | 
| 테스트용 복사 인스턴스 생성 | 원본 구성을 기준으로 생성 |
| 유사 운영 환경 스테이징 인스턴스 | 운영 인스턴스와 설정 유사 |
| 특정 환경 기반 재현 테스트 | 동일 AMI + 네트워크 구성 |
