## 핵심 요약

- Launch More Like This는 기존 EC2 인스턴스의 구성 정보를 기준 템플릿처럼 활용하여 빠르게 새 인스턴스를 시작하는 기능
- 기존 인스턴스를 참조 원본으로 하여 AMI, 인스턴스 유형, 네크워크, 스토리지, 태그 등을 자동으로 불러오고 수정해 시작할 수 있음
- 템플릿 생성 없이도 일회성으로 유사 구성 인스턴스 빠르게 생성 가능

---

## 기능 개요

- AWS 콘솔 EC2 인스턴스 목록에서
  - 기존 인스턴스 선택
  - Actions ➡️ Launch More Like This 선택
  - 기존 구성값을 기반으로 새 인스턴스 설정화면이 열림
- 기존 인스턴스 구성 자료를 자동으로 불러와 재사용/수정/적용함

---

## 작동 원리

- 기존 인스턴스 정보 수집
  - EC2 콘솔이 선택한 인스턴스의 시작 파라미터를 기반으로 설정값을 추출
- Launch More Like This 화면 오픈
  - 해당 구성값을 인스턴스
