## EC2에서 AWS Marketplace AMI로 인스턴스 시작

#### 핵심 요약

- AWS Marketplace는 EC2에서 실행할 수 있는 상용/서드파티 소프트웨어 AMI를 제공하는 마켓
- Marketplace AMI를 사용하려면 먼저 제품 구독 후 인스턴스를 시작해야 함
- 인스턴스 시작 후 Running 상태 동안 요금이 발생하며 일부 AMI는 소프트웨어 사용료가 추가됨

---

## 1. AWS Marketplace AMI 개념

#### AWS Marketplace AMI

- EC2에서 실행 가능한 소프트웨어 포함 이미지
- AWS 또는 서드파티 벤더가 제공
- 운영체제 + 애플리케이션 포함된 형태

<br>

#### 예시

- 보안솔루션
- DB 서버
- 개발 툴
- 모니터링 시스템
