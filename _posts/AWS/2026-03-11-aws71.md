## 핵심 요약

- Linux EC2 인스턴스는 SSH 프로토콜을 사용해 접속
- 접속을 위해 필요한 핵심요소
  - EC2 Key Pair(.pem)
  - 퍼블릭 IP 또는 퍼블릭 DNS
  - SSH 클라이언트
  - AMI에 맞는 사용자 계정
- 기본 SSH 접속 명령 예:

```bash
ssh -i my-key.pem ec2-user@<EC2-Public-IP>
```
