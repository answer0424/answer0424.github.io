## 핵심 요약

- Windows에서 EC2 Linux 인스턴스에 접속하여면 SSH 클라이언트가 필요
- AWS에서는 OpenSSH 또는 PuTTY 사용을 안내
- 접속에 필요한 핵심 요소
  - EC2 Key Pair(.pem)
  - 퍼블릭 IP 또는 퍼블릭 DNS
  - PuTTY SSH 클라이언트
  - AMI에 맞는 사용자 계정

---

## 1. Windows에서 사용할 수 있는 SSH 방법

| 방법 | 특징 |
|-|-|
| OpenSSH | Windows 기본 SSH |
| PuTTY | Windows 전용 SSH 클라이언트 |
| WSL + SSH | Linux 환경에서 SSH 사용 |
