## 핵심 요약

- EC2 Windows 인스턴스는 RDP 를 사용해서 접속
- 접속을 위해 필요한 핵심 요소는 다음과 같음
1. Windows EC2 인스턴스
2. 퍼블릭 IP 또는 퍼블릭 DNS
3. 관리자 계정 암호
4. RDP 클라이언트
