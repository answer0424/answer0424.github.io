## 핵심 요약

- EC2 Windows 인스턴스는 RDP 를 사용해서 접속
- 접속을 위해 필요한 핵심 요소는 다음과 같음
1. Windows EC2 인스턴스
2. 퍼블릭 IP 또는 퍼블릭 DNS
3. 관리자 계정 암호
4. RDP 클라이언트

---

## 1. 사전 조건

#### 인스턴스 상태

> 인스턴스는 반드시 Running 상태여야 함

- 확인 위치

```text
EC2 Console ➡️ Instances ➡️ Instance State
```

<br>

#### 보안 그룹 설정

> RDP 접속을 위해 3389 포트가 허용되어야 함

| 설정 | 값 |
|-|-|
| Protocol | TCP |
| Port | 3389 |
| Source | 접속 IP |

<br>

#### 퍼블릭 IP/DNS

- Public IPv4
- Public DNS (IPv4)

> 퍼블릭 IP가 없다면 Elastic IP 연결이 필요할 수 있음

---

## 2. 관리자 비밀번호 가져오기

> EC2 Windows 인스턴스는 처음 접속할 때 Administrator 암호를 복호화해야 함

#### 절차

1. EC2 콘솔 접속
2. Instances 선택
3. Connect 선택
4. RDP client 탭 선택
5. Get Password 클릭
6. Key Pair `.pem` 파일 업로드
7. Decrypt Password

---

## 3. RDP 클라이언트로 연결

#### RDP 파일 다운로드

- Connect 페이지에서 
  
```text
Download Remote Desktop File
```

- 파일 다운로드 후 실행함
