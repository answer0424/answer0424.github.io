#### getKakaoCertStatus

> 카카오페이 인증상태 조회

- 입력 파라미터에서 `tx_id`를 추출
- 추출한 `tx_id`를 쿼리 파라미터로 설정하고 HTTP GET 요청
- GET 요청으로 API 호출
- 응답 본문을 UFT-8로 읽어 `result`에 저장
- 응답 `JSON`을 `Map`으로 변환
- 이중 척번째 signed_data 항목을 `rtnMap`으로 리턴
