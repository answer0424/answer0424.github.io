## 관계 연산자

> 관계 연산자는 두 수의 관계를 비교하여 참 또는 거짓을 결과로 얻는 연산자

- 거짓은 0, 참은 1로 사용되지만 0외의 모든 숫자도 참으로 간주됨

| 연산자 | 의미 |
| --- | --- |
| == | 같다 |
| != | 같지 않다 |
| > | 크다 |
| < | 작다 |
| >= | 크거나 같다 |
| <= | 작거나 같다 |

## 비트 연산자 

| 연산자 | 의미 | 비고 |
| --- | --- | --- |
| & | 비트 AND 연산자 | 두 비트가 모두 1일 때만 1을 결과로 얻음 |
| \| | 비트 OR 연산자 | 두 비트 중 하나라도 1이면 1을 결과로 얻음 | 
| ^ | 비트 XOR 연산자 | 두 비트가 서로 다를 때만 1을 결과로 얻음 |
| ~ | 비트 NOT 연산자 | 비트를 반전시킴 |
| << | 왼쪽 시프트 연산자 | 지정한 비트 수만큼 왼쪽으로 이동시킴 |
| >> | 오른쪽 시프트 연산자 | 지정한 비트 수만큼 오른쪽으로 이동시킴 |
