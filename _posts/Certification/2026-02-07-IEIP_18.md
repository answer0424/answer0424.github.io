## HTML의 개요

> HTML(HyperText Markup Language)은 인터넷 표준 문서인 하이퍼텍스트 문서를 만들 때 사용하는 마크업 언어임

- 웹 브라우저에 표시되는 화면이 바로 HTML로 작성된 문서
- HTML은 태그로 구성되는데 상위 태그 아래에 여러 개의 하위 태그가 있는 트리 구조
- 태그는 속성을 사용하여 기능을 구체화할 수 있으며, 대소문자를 구분하지 않음
- 속성에 값을 지정할 때는 일반적으로 큰 따옴표로 묶음
- 값을 작은 따옴표로 묶거나 그냥 값만 입력할 수도 있지만, 값에 띄어쓰기가 포함된 경우에는 반드시 따옴표로 묶어야 함

---

## HTML의 기본 구조

```html
<!DOCTYPE html>
<html> 
<head>
    <meta charset="UTF-8">
    <title>문서 제목</title>
</head>
<body>
    문서 내용 작성
</body>
</html>
```

## 목록 태그

#### `<ul>`

- 순서 없는 목록을 표시
- 순서 없이 항목 앞에 점(•)이 표시됨
- <li> 태그를 이용해 목록을 작성

<br>

#### `<ol>`

- 순서 있는 목록을 표시
- 항목 앞에 일련번호를 붙여 표시
- <li> 태그를 이용해 목록을 작성

<br>

#### `<dl>`

- 제목 있는 목록을 표시
- <dt> 태그를 이용해 제목을 작성
- <dd> 태그를 이용해 제목의 하위 내용을 작성

---

## 프레임

#### `<frameset>`

- 화면을 분할함
- 태그 사이에는 분할한 프레임의 개수만큼 `<frame>` 태그를 사용

<br>

#### `<frame>`

- 분할한 각가의 프레임에 표시할 HTML 문서를 지정
- `<frameset>` 으로 분할한 영역에 `<frame>` 태그가 적용되는 순서
  - 화면을 가로로 분할한 경우: 왼쪽 ➡️ 아래쪽
  - 화면을 세로로 분할한 경우: 위 ➡️ 오른쪽
- 분할한 프레임의 개수는 `<frameset>`의 rows 또는 cols 속성으로 알 수 있음
