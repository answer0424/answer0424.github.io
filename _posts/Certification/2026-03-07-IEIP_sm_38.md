## SQL의 개념

#### DDL(Data Definition Language)

- SCHEMA, DOMIAN, TABLE, VIEW, INDEX를 정의하거나 변경 또는 삭제할 떄 사용하는 언어
- DDL(데이터정의어)의 세 가지 유형

| 명령어 | 기능 |
| --- | --- |
| CREATE | SCHEMA, DOMAIN, TABLE, VIEW, INDEX를 정의 |
| ALTER | TABLE에 대한 정의를 변경하는 데 사용함 |
| DROP | SCHEMA, DOMAIN, TBALE, VIEW, INDEX를 삭제 |

<br>

#### DML(Data Manipulation Language)

- 데이터베이스 사용자가 응용 프로그램이나 질의어를 통하여 저장된 데이터를 실질적으로 처리하는 데 사용되는 언어
- DLM(데이터조작어)의 세 가지 유형

| 명령어 | 기능 |
| --- | --- |
| SELECT | 테이블에서 조건에 맞는 튜플을 검색 |
| INSERT | 테이블에 새로운 튜플을 삽입 |
| UPDATE | 테이블에 조건에 맞는 튜플의 내용을 변경 |
| DELETE | 테이블에 조건에 맞는 튜플을 삭제 |
