## 43번

#### DBMS의 제어 기능이 갖추어야 할 요건에 해당하지 않는 것은?

- 갖추어야 할 요건
  - 데이터베이스를 접근하는 갱신, 삽입, 삭제 작업이 정확하게 수행되게 해야 함
  - 정단한 사용자가 허가된 데이터만 접근할 수 있도록 보안을 유지하여야 함
  - 여러 사용자가 데이터베이스를 동시에 접근하여 처리할 때 데이터베이스와 처리 결과가 항상 정확성을 유지하도록 병행 제어를 할 수 있어야 함
- 아닌 것: 데이터와 데이터의 관계를 명확하게 명세할 수 있어야 하며, 원하는 데이터 연산은 무엇이든 명세할 수 있어야 함

<br>

## 46번

#### Preoder 운행

- preorder: root ➡️ left ➡️ right
- inorder: left ➡️ root ➡️ right
- postorder: left ➡️ right ➡️ root

## 50번

#### 키는 개체 집합에서 고유하게 개체를 식별할 수 있는 속성, 데이터베이스에서 사용되는 키의 종류에 대한 설명 중 옳지 않은 것?

- 옳은 것
  - 후보키는 개체들을 고유하게 식별할 수 있는 속성
  - 슈퍼키는 한 릴레리션 내의 속성들의 집합으로 구성된 키
  - 외래키는 다른 릴레이션의 기본키를 참조하는 속성 또는 속성들의 집합
- 아닌 것
  - 보조키는 후보키 중에서 대표로 선정된 키
- 정답: 4번

<br>

## 56번

#### 관계 데이터베이스의 구성 요소에 대한 설명 중 가장 옳지 않은 것

- 옳은 것

  - 속성은 릴레이션을 구성하는 항목
  - 각 속서은 릴레이션 내에서 유일한 이름을 가짐
- 틀린 것
  - 하나의 릴레이션을 구성하는 튜플은 모두 같음
- 정답: 3번
