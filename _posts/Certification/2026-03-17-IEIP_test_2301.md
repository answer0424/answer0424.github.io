## 4번

#### OSI 7계층 중 데이터 링크 계층의 프로토콜에 해당하지 않는 것은?

- PPP
- HDLC
- LLC
- 옳지 않은 것
  - UDP
- 해설: 데이터 링크 계층은 2계층으로 주요 역할은 물리적 링크를 통한 데이터 전송, 오류 검출, 프레임 단위 전송 등(PPP, HDLC, LLC, Ethernet)
  - UDP는 4계층인 전송 계층 프로토콜임
- 정답: 4번

## 20번

#### 결합도의 세기

- 결합도는 모듈 간의 상호 의존성 정도를 나타내는 지표로, 낮을수록 좋음
- 결합도의 세기는 다음과 같이 분류됨
- 내용 결합 ➡️ 공통 결합 ➡️ 외부 결합 ➡️ 제어 결합 ➡️ 스탬프 결합 ➡️ 자료 결합 (내공외제스자)

<br>

## 21번

#### 한 모듈이 다른 모듈의 내부 자료를 직접적으로 참조하는 경우의 결합도

- 내용 결합도: 한 모듈이 다른 모듈의 내부 데이터나 로직을 직접 참조
- 공통 결합도: 여러 모듈이 공통 전역 데이터를 공유
- 외부 결합도: 외부 장치, 파일, 통신 프로토콜 등을 공유
- 제어 결합도: 한 모듈이 다른 모듈의 흐름을 제어하는 값을 전달
- 스탬프 결합도: 자료 구조 전체를 전달하지만 일부만 사용
- 자료 결합도: 필요한 데이터만 매개변수로 전달
- 정답: 1번

<br>

## 22번

#### 특정 모듈에 대해서 존재하는 처리 요소들 간의 기능적 연관성을 의미하는 것으로 입력이나 에러 처리 같은 유사한 기능을 행하는 요소끼리 하나의 요소로 묶는 응집도?

- 논리적 응집도: 유사한 기능을 논리적으로 묶음
- 옳지 않은 것
  - 교환적 응집도: 존재 X
  - 순차적 응집도: 한 단계의 출력이 다음 단계의 입력이 되는 구조
  - 절차적 응집도: 같은 절차 흐름에 따라 수행되는 기능
- 해설: 응집도는 높을수록 좋으며 모듈 내부 요소들 간의 관련성 정도를 의미
  - 응집도의 세기: 우연적 ➡️ 논리적 ➡️ 시간적 ➡️ 절차적 ➡️ 통신적 ➡️ 순차적 ➡️ 기능적 (우논시절통순기)
- 정답: 3번

## 41번

#### 릴레이션

- 릴레이션(Relation) - 테이블
- 튜플 (tuple) - 행(row, record)
- 속성(Attribute) - 열(column, field)
- 차수(degree) - 속성의 수(열의 개수)
- 카디널리티(Cardinality) - 튜플의 개수(행의 개수)
- 도메인(domain) - 속성이 가질 수 있는 값의 집합
