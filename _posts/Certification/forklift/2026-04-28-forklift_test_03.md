## 기계식 분사펌프가 장착된 디젤기관에서 가동 중에 발전기가 고장이 났을 때 발생할 수 있는 현상으로 틀린 것은?

1. 충전경고등이 들어온다
2. 배터리가 방전되어 시동이 꺼지게 된다
3. 헤드램프를 켜면 불빛이 어두워진다
4. 전류계의 지침이 (-)쪽을 가리킨다

> 2번 / 발전기 고장은 충전이 안됨을 뜻한다. 충전이 안된다고 바로 배터리가 방전되고 꺼지지는 않는다

## 연료탱크의 연료를 분사펌프 저압부까지 공급하는 것은?

1. 연료공급 펌프
2. 연료분사 펌프
3. 인젝션 펌프
4. 로터리 펌프

> 1번 / 연료탱크 연료는 연료공급펌프를 통해 분사펌프 저압부까지 공급됨
>
> 연료의 공급 경로
>
> 연료탱크 → 연료공급펌프 → 연료필터 → 분사펌프 → 분사노즐

<br>

## 오일의 압력이 높은 것과 관계없는 것은?

1. 릴리프 스프링이 강할 때
2. 추운 겨울철에 가동할 때
3. 오일 점도가 높을 때
4. 오일 점도가 낮을 때

> 4번 / 점도는 오일의 묽기이므로 점도가 낮으면 압력이 낮고, 점도가 높으면 압력이 높다(점도와 오일 압력은 비례관계)

<br>

## 반도체에 대한 설명 중 틀린 것은?

1. 반도체는 양도체와 절연체의 중간 범위임
2. 고유저항은 10-3~10-4Ω 정도임
3. 실리콘, 게르마늄, 셀렌 등이 있음
4. 절연체의 설징을 띠고 있음

> 4번 / 반도체는 양의 극성을 가진 도체인 양도체와 전기 전달이 어려운 절연체의 중간 범위의 성질을 띤다
