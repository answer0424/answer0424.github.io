## 8번

#### TCP/IP의 IP Layer에 해당하는 프로토콜은?

- ICMP
- 옳지 않은 것
  - SMTP
  - HTTP
  - UDP

```textplain
# TCP/IP 모델은 일반적으로 4계층으로 구성됨

1. Application Layer - 응용 서비스 제공(HTTP, SMTP, FTP, DNS)
2. Transport Layer - 종단 간 통신(TCP, UDP)
3. Internet Layer - 논리적 주소 및 라우팅(IP, ICMP, ARP)
4. Link Layer - 실제 네트워크 전송(Ethernet, Wi-Fi)
```

## 34번

#### 예외 처리에 대한 설명으로 옳지 않은 것은?

- 예외가 발생했을 때 프로그래머가 해당 문제에 대비해 작성해 놓은 처리 루틴을 수행하도록 하는 것을 예외 처리하고 함
- catch 블록에서 선언한 변수는 해당 catch 블록에서만 유효함
- try ~ catch 문 안에 또다른 try ~ catch문이 올 수 있음
- 옳지 않은 것
  - C++에서는 try, catch, finally를 이용하여 예외처리를 수행함(finally는 Java에서 사용되는 키워드로, C++에서는 존재하지 않음)
- 정답: 1번

## 39번

#### 다음이 설명하는 응집도의 유형은?

```text
모듈이 다수의 관련 기능을 가질 때 모듈 안의 구성 요소들이 그 기능을 순차적으로 수행할 경우의 응집도
```

- 절차적응집도
- 옳지 않은 것

  - 우연적 응집도 ➡️ 모듈의 구성 요소들이 서로 관련이 없는 경우
  - 논리적 응집도 ➡️ 모듈이 여러 관련 기능을 가지고 있지만, 그 기능들이 서로 독립적인 경우

  - 캡슐화는 상위 클래스에서 속성이나 연산을 전달받아 새로운 형태의 클래스로 확장하여 사용하는 것을 의미(상속의 설명)
