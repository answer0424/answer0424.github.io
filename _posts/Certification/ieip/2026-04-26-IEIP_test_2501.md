## OSI 7계층 중 IP 프로토콜을 기반으로 하며, 통신망 연결을 수행하는 기능을 하는 계층은?

1. 물리 계층
2. 데이터 링크 계층
3. 네트워크 계층
4. 전송 계층

> 3번
>
> 물리 계층: 전송에 필요한 두 장치 간의 실제 접속과 절단 등 기계적, 전기적, 기능적, 절차적 특성에 대한 규칙을 정의함
>
> 데이터 링크 계층: 두 개의 인접한 개발 시스테들 간에 신뢰성 있고 효율적인 정보 전송을 할 수 있도록 시스템 간 연결 설정과 유지 및 종료를 담당
> 전송 계층: 논리적 안정과 균일한 데이터 전송 서비스를 제공함으로써 종단 시스템 간에 투명한 데이터 전송을 가능하게 함

## 한 객체의 상태가 변화하면 객체에 상속되어 있는 다른 객체들에게 변화된 상태를 전달하는 패턴은?

1. State
2. Observer
3. Visitor
4. Mediator

> 2번
>
> State: 객체의 상태에 따라 동일한 동작을 다르게 처리해야 할 때 사용하는 패턴
>
> Visitor: 각 클래스들의 데이터 구조에서 처리 기능을 분리하여 별도의 클래스로 구성하는 패턴
>
> Mediator: 수많은 객체들 간의 복잡한 상호작용을 캡슐화하여 객체로 정의하는 패턴

## 다음에서 설명하고 있는 관계는?

```text
- 하나의 사물이 다른 사물에 비해 더 일반적인지 구체적인지를 표현
- 일반적인 개념을 상위, 보다 구체적인 개념을 하위라고 부르며 구체적인 사물에서 일반적인 사물 쪽으로 속이 빈 화살표를 연결하여 표현
```

1. 일반화
2. 포함
3. 연관
4. 집합

> 1번
>
> 포함: 집합 관계의 특수한 형태로, 포함하는 사물의 변화가 포함되는 사물에게 영향을 미치는 관계를 표현
>
> 연관: 2개 이상의 사물이 서로 관련되어 있음을 표현
> 
> 
> 집합: 하나의 사물이 다른 사물에 포함되어 있는 관계를 표현

## IPv6에 대한 설명으로 틀린 것은?

1. IPv6 주소는 128비트로 구성됨
2. 인증 및 보안 기능을 포함하고 있음
3. 브로드캐스트, 유니캐스트, 멀티캐스트로 구성
4. IPv6 헤더는 9새의 필드로 구성

> 3번 / 유니캐스트, 멀티캐스트, 애니캐스로 구성
