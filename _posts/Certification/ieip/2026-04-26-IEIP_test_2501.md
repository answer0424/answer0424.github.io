## OSI 7계층 중 IP 프로토콜을 기반으로 하며, 통신망 연결을 수행하는 기능을 하는 계층은?

1. 물리 계층
2. 데이터 링크 계층
3. 네트워크 계층
4. 전송 계층

> 3번
>
> 물리 계층: 전송에 필요한 두 장치 간의 실제 접속과 절단 등 기계적, 전기적, 기능적, 절차적 특성에 대한 규칙을 정의함
>
> 데이터 링크 계층: 두 개의 인접한 개발 시스테들 간에 신뢰성 있고 효율적인 정보 전송을 할 수 있도록 시스템 간 연결 설정과 유지 및 종료를 담당
> 전송 계층: 논리적 안정과 균일한 데이터 전송 서비스를 제공함으로써 종단 시스템 간에 투명한 데이터 전송을 가능하게 함

## 한 객체의 상태가 변화하면 객체에 상속되어 있는 다른 객체들에게 변화된 상태를 전달하는 패턴은?

1. State
2. Observer
3. Visitor
4. Mediator

> 2번
>
> State: 객체의 상태에 따라 동일한 동작을 다르게 처리해야 할 때 사용하는 패턴
>
> Visitor: 각 클래스들의 데이터 구조에서 처리 기능을 분리하여 별도의 클래스로 구성하는 패턴
>
> Mediator: 수많은 객체들 간의 복잡한 상호작용을 캡슐화하여 객체로 정의하는 패턴

<br>

## 다음에서 설명하고 있는 관계는?

```text
- 하나의 사물이 다른 사물에 비해 더 일반적인지 구체적인지를 표현
- 일반적인 개념을 상위, 보다 구체적인 개념을 하위라고 부르며 구체적인 사물에서 일반적인 사물 쪽으로 속이 빈 화살표를 연결하여 표현
```

1. 일반화
2. 포함
3. 연관
4. 집합

> 1번
>
> 포함: 집합 관계의 특수한 형태로, 포함하는 사물의 변화가 포함되는 사물에게 영향을 미치는 관계를 표현
>
> 연관: 2개 이상의 사물이 서로 관련되어 있음을 표현
> 
> 
> 집합: 하나의 사물이 다른 사물에 포함되어 있는 관계를 표현

## 파일 조직 기법 중 순차 파일에 대한 설명으로 옳지 않은 것은?

1. 파일 탐색 시 효율이 우수하며, 대화형 처리에 적합
2. 레코드가 키 순서대로 편성되어 취급이 용이
3. 연속적인 레코드의 저장에 의해 레코드 사이에 빈 공간이 존재하지 않으므로 기억장치의 효율적인 이용이 가능
4. 필요한 레코드를 삽입, 삭제, 수정하는 경우 파일을 재구성해야 하므로 파일 전체를 복사해야 함

> 3번

## 자원 보호 기법 중 접근 제어 행렬에서 수평으로 있는 각 행들만을 따온 것으로서 각 영역에 대한 권한은 객체와 그 객체에 허용된 연산자로 구성되는 것은?

1. Global Table
2. Access Control List
3. Capability List
4. Lock/Key

> 3번 
>
> Global Table: 가장 단순한 구현 방법으로 세 개의 순서쌍인 영역, 객체, 접근 권한의 집합을 목록 형태로 구성한 기법
>
> Access Control List: 접근 제어 행렬에 있는 각 열, 즉 객체를 중심으로 접근 리스트를 구성한 기법
>
> Lock/Key: 접근 제어 리스트와 권한 리스트를 절충한 기법으로 각 객체는 Lock, 각 영역은 Key라 불리는 유일하고도 독특한 값을 갖고 있어서 영역과 객체가 일치하는 경우에만 해당 객체를 접근할 수 있음

<br>
