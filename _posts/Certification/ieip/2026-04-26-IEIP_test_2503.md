## IPv6 에 대한 설명으로 틀린 것은?

1. IPv6 주소는 128비트로 구성
2. 유니캐스트, 멀티캐스트, 애니캐스트를 지원
3. 주소를 32비트씩 나눠서 8진수로 쓰고 마침표로 구분
4. 프로토콜의 확장을 허용하도록 설계되었음

> 3번 / IPV6은 16비트 씩 8부분으로 구성되는데 각 부분을 16 진수로 표현하고, 콜론(:)으로 구분함

## 유스케이스 다이어그램의 용어 중 프로젝트 개발 범위에 속하지 않고, 이미 다른 프로젝트에서 개발되어 사용중인 것으로 원래 프로젝트와 연동이 되는 또 다른 시스템을 무엇이라고 하는가?

1. System Scope
2. User Actor
3. Use Case
4. System Actor

> 4번 
>
> System Scope: 시스템 내부에서 수행되는 기능들을 외부 시스템과 구분하기 위해 시스템 내부의 유스케이스들을 사각형으로 묶어 시스템의 범위를 정함
>
> User Actor: 시스템을 사용함으로써 이득을 얻는 대상으로 주로 사람이 해당
>
> Use Case: 사용자가 보는 관점에서 시스템이 액터에게 제공하는 서비스 또는 기능을 표현한 것
>
> System Actor: 프로젝트 개발 범위에 속하지 않고, 이미 다른 프로젝트에서 개발되어 사용중인 것

<br>

## C언어에서 문자열을 화면에 출력할 때 사용하는 함수는?

## 보안 취약점이 발견되었을 떄 발견된 취약점의 존재 자체가 널리 공표되기도 전에 해당 취약점을 통하여 이루어지는 보안 공격은?

1. DDoS 공격
2. 제로 데이 공격
3. 사회 공학적 공격
4. Land 공격

> 2번 
>
> DDoS 공격: 여러 곳에 분산된 공격 지점에서 한 곳의 서버에 대해 분산 서비스 공격을 수행하는 것
>
> 사회 공학: 컴퓨터 보안에 있어서 인간 상호 작용의 깊은 신뢰를 바타으로 사람들을 속여 보안 절차를 꺠트리기 위한 비기술적 시스템 침입 수단
>
> Land 공격: 패킷을 전송할 때 송신 IP 주소와 수신 IP 주소를 모두 공격 대상의 IP 주소로 하여 공격 대상에게 전송하는 것
