# 1과목

## 운영체제 운용 기법

1. 일괄처리: 일정량 또는 일정기간 동안 데이터를 모아서 한꺼번에 처리
2. 실시간처리: 데이터 발생 즉시 처리하여 결과를 산출하는 방식
3. 다중 프로그램: 하나의 CPU와 주기억장치를 이용하여 여러 개의 프로그램을 동시에 처리
4. 다중 처리: 여러개의 CPU와 주기억장치를 이용
5. 시분할: 여러명의 사용자가 사용하는 시스템에서 번갈아가면서 처리(라운드로빈 방식이라고도 함)
