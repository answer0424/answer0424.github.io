## 5번

#### Lan의 한 정류인 100Base-T 네트워크에서 사용되는 전송 매체는?

- UTP cable
- 옳지 않은 것
  - Coaxial cable: 초기 Ethernet에서 사용
  - Optical cable: 100Base-FX 등 광케이블 규격
  - Microwave cable: LAN에서 사용하지 않음
- 해설: 구성의 의미
  - 100 ➡️ 전송 속도 100Mbps
  - Base ➡️ Baseband 전송 방식
  - T ➡️ Twisted Pair 케이블 사용
- 정답: 3번

## 38번

#### C언어에서 정수 변수 a, b에 각각 1, 2가 저장되어 있을 때 다음 식의 연산 결과로 옳은 것은?

```C
a < b + 2 && a << 1 <= b
```

- 해설
  - 연산자 우선 순위: `+`, `<<`, `<` `<=`, `&&`

```C
(a < (b + 2)) && ((a << 1) <= b)
```
- 정답: 1

## 56번

#### 데이터베이스 설계 단계 중 물리적 설계에 대한 설명으로 옳지 않은 것은?

- 다양한 데이터베이스 응용에 대해서 처리 성능을 얻기 위해 데이터베이스 파일의 저장 구조 및 액세스 경로를 결정
- 물리적 저장장치에 저장할 수 있는 물리적 구조의 데이터로 변환하는 과정
- 물리적 설계에서 옵션 선택 시 응답시간, 저장 공간의 효율화, 트랜잭션 처리율 등을 고려해야 함
- 옳지 않은것
  - 개념적 설계 단계에서 만들어진 정보 구조로부터 특정 목표 DBMS가 처리할 수 있는 스키마를 생성함
