## Python의 활용

- 객체지향 기능을 지원하는 인터프리터 언어로 다른 언어에 비해 문법이 간단

<br>

#### Python과 C, Java의 차이점

- 변수의 자료형에 대한 선언이 없음
- 문자와 문자열을 작은따옴표, 큰따옴표 구분 없이 사용할 수 있음
- 변수에 연속하여 값을 저장하는 것이 가능
- 문장 끝을 의미하는 세미콜론의 사용이 불필요
- if나 for과 같이 코드 블록을 포함하는 명령문을 작성할 때 중괄호 대신 콜론과 여백으로 구분

## Range

- Range는 연속된 숫자를 생성하는 것으로 리스트나 반복문에서 많이 사용
- 사용법
  - range(최종값): 0에서 '최종값'-1까지 연속된 숫자를 생성
  - range(초기값, 최종값): '초기값'에서 '최종값'-1까지 연속된 숫자를 생성
  - range(초기값, 최종값, 증가값): '초기값'에서 '최종값'-1까지 '증가값'만큼 증가하면서 숫자를 생성

## Slice

- 슬라이스는 문자열이나 리스트와 같은 순차형 객체에서 일부를 잘라 반환하는 기능
- 사용법
  - 객체명[초기위치:최종위치] : '초기위치'에서 '최종위치'-1까지의 모든 요소를 가져옴
  - 객체명[초기위치:최종위치:증가값] : '초기위치'에서 최종위치'-1까지 증가값만큼 증가하면서 해당 위치의 요소들을 가져옴
  - 객체명[:]또는 객체명[::] : 객체의 모든 요소를 가져옴
  - 객체명[초기위치:] : 객체의 '초기위치에서 마지막 위치까지 모든 요소를 가져옴
  - 객체명[:최종위치] : 객체의 0번째 위치에서 '최종위치'-1까지 모든 요소를 가져옴
  - 객체면[::증가값] : 객체의 0번째 위치에서 마지막위치까지 '증가값'만큼 증가하면서 해당 요소들을 가져옴

## 리스트 관련 주요 메서드

- pop(위치): 리스트의 위치에 있는 값을 출력하고 해당 요소를 삭제
- index(값): 리스트에서 값이 저장된 요소의 위치를 반환
- count(값): 리스트에서 값이 저장되어 있는 요소들의 개수를 반환
- extend(리스트): 리스트의 끝에 새로운 리스트를 추가하여 확장
- reverse(): 리스트의 순서를 역순으로 뒤집음
- len(): 리스트의 길이를 반환
- sort(): 리스트를 정렬하며, 기본값을 오름차순임 / sort(reverse = true)는 내림차순
- copy(): 리스트를 복사
