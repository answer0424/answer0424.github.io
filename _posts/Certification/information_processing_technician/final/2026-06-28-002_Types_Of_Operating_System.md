## Windows

#### 그래픽 사용자 인터페이스(GUI)

- 마우스로 아이콘이나 메뉴를 선택하여 작업

<br>

#### 선점형 멀티태스킹

- 동시에 여러 개의 프로그램을 실행하는 멀티태스킹을 하면서 운영체제가 각 작업의 CPU 이용 시간을 제어하여 응용 프로그램 실행 중 문제가 발생하면 해당 프로그램을 강제 종료시키고 모든 시스템 자원을 반환

<br>

#### PnP(자동 감지 기능)

- 컴퓨터 시스템에 프린터나 사운드카드 등의 하드웨어를 설치했을 떄, 해당 하드웨어를 상ㅇ하는 데 필요한 시스템 환경을 운영체제가 자동으로 구성해주는 기능

## UNIX

- 시분할 시스템을 위해 설계된 대화식 운영체제
- 개방형 시스템
- 대부분 C언어로 작성되어 있어 이식성이 높고, 프로세스 간의 호환성이 높음
- 다중 사용자, 다중 작업을 지원
- 트리 구조의 파일 시스템을 가짐

---

## UNIX 시스템의 구성

#### 커널(Kernel)

- 하드웨어를 보호하고, 프로그램과 하드웨어 간의 인터페이스 역할
- 가장 핵심적인 부분으로 컴퓨터가 부팅될 때 주기억장치에(RAM) 상주

<br>

#### 쉘(Shell)

- 사용자의 명령어를 인식하여 프로그램을 호출하고 명령을 수행하는 명령어 해석기
- 시스템과 사용자의 인터페이스를 담당
- 명령어가 포함된 파일 형태로 보조기억장치에 존재하며, 필요할 때 주기억장치에 적재
- 종류:  Bourne Shell, C Shell, Korn Shell

<br>

#### 유틸리티 프로그램

- 일반 사용자가 작성한 응용 프로그램을 처리하는 데 사용
- DOS에서의 외부 명령어에 해당
