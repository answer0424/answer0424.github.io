## 하드웨어 환경

#### 웹서버

- HTTP 프로토콜을 통해 클라이언트로부터 직접 요청을 받아 처리한 후 결과를 클라이언트에 다시 보냄
- 저용향의 정적 파일을 제공

<br>

#### 웹 애플리케이션 서버

- 동적 서비스를 제공하거나, 웹 서버와 데이터베이스 서버 또는 웹 서버와 파일 서버에서 인터페이스 역할을 수행

<br>

#### 데이터베이스 서버

- 데이터베이스와 이를 관리하는 DBMS를 운영

<br>

#### 파일 서버

- 데이터베이스에 저장하기에는 비효율적이거나, 서비스 제공을 목적으로 유지하는 파일들을 저장
