## DCL(Data Control Language, 데이터 제시어)

- 데이터의 보안, 무결성, 회복 병행 제어 등을 정의하는 데 사용하는 언어
- DCL의 종류

| 명령어 | 기능 |
|-|-|
| COMMIT | 명령에 의해 수행된 결과를 실제 무리적 디스크로 저장하고, 데이터베이스 조작 작업이 정상적으로 완료되었음을 알림 |
| ROLLBACK | 데이터베이스 조작 작업이 비정상적으로 종료되었을 때 원래의 상태로 복구함 |
| GRANT | 데이터베이스 사용자에게 사용권한을 부여 |
| REVOKE | 데이터베이스 사용자의 사용 권한을 취소 |
