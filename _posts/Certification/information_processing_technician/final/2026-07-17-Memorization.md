## 운영체제 운용 기법

1. 일괄 처리 시스템: 데이터를 모아서 한번에 처리
2. 다중 프로그래밍 시스템: 하나의 CPU와 주기억장치를 이용해 여러 개의 프로그램을 동시 처리
3. 시분할 시스템: 라운드 로빈이라고 하며, 모든 작업이 동시에 진행되는 것처럼 보임
4. 다중 처리 시스템: 여러 개의 CPU와 주기억장치를 이용해 여러 개의 프로그램을 동시 처리
5. 실시간 처리 시스템: 데이터 발생 즉시 처리
6. 다중 모드 처리: 일괄 처리, 시분할, 다중 처리, 실시간 처리를 한번에 모아둔 시스템
7. 분산 처리 시스템: 여러 개의 컴퓨터를 통신 회선으로 연결하여 하나의 작업을 처리

## 스케줄링

#### 비선점: 이미 할당된 CPU를 빼앗아 사용할 수 없는 기법

- FCFS: 도착한 순서대로 처리
- SJF: 실행 시간이 가장 짧은 순서대로 처리
- HRN: 우선순위를 계산해서 가장 높은 순서대로 처리 -> (대기시간 + 실행시간) / 실행시간
