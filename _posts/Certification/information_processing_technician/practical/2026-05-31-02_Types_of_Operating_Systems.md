## Windows

- GUI: 키보드로 명령어를 직접 입력하지 않고 마우스로 모든 작업을 수행하는 방식
- 선점형 멀티태스킹: 동시에 여러 개의 프로그램을 실행하는 멀티태스킹을 하면서 운영체제가 각 작업을 CPU 이용 시간을 제어하여 응용 프로그램 실행 중 문제가 발생하면 해당 프로그램을 강제로 종료시키고 모든 시스템 자원을 반환하는 방식 (여러가지 프로그램을 실행하면서 도중에 강제 종료시킬 수 있음)
- PnP: 자동 감지 기능(USB, 헤드셋)
- OLE: 다른 여러 응용 프로그램에서 작성된 문자나 그림 등의 개체를 현재 작성 중인 문서에 자유롭게 연결하거나 삽입 가능
- Single-User 시스템

## UNIX

- 시분할 시스템을 위해 설계된 대화식 운영체제
- 대부분 C언어로 작성되어 있어 이식성이 높으며 장치, 프로세스 간의 호환성이 높음
- 다중 사용자, 다중 작업을 지원
- 트리 구조의 파일 시스템을 갖음 

#### UNIX 시스템의 구성

##### 커널

- 하드웨어를 보호하고, 프로그램과 하드웨어 간의 인터페이스 역할을 담당
- UNIX의 가장 핵심적인 부분

<br>

##### 쉘

- 사용자의 명령어를 인식하여 프로그램을 호출하고 명령을 수행하는 명령어 해석기
- 시스템과 사용자 간의 인터페이스를 담당
- 종류: Bourne Shell, C Shell, Korn Shell

<br>

##### 유틸리티 프로그램

- 일반 사용자가 작성한 응용 프로그램을 처리하는 데 사용함
- DOS에서의 외부 명령어에 해당됨
- 종류: 에디터, 컴파일러, 인터프리터, 디버거 등

---

## LINUX

- 리누스 토발즈가 UNIX를 기반으로 개발한 운영체제
- 프로그램 소스 코드가 무료로 공개되어 있기 때문에 프로그래머가 원하는 기능을 추가할 수 있음
- UNIX와 완벽하게 호환됨

--- 

## MacOS

- 애플 사가 UNIX를 기반으로 개발한 운영체제
- 드라이버 설치 및 install과 uninstall과정이 단순

---

## Android

- 구글 사에서 개발한 리눅스 커널 기반의 개방형 모바일 운영체제
- 모든 코드가 공개된 개방형 소프트웨어
- 자바와 코들린으로 애플리케이션을 작성

---

## iOS

- 애플 사에서 개발한 UNIX 기반의 모바일 운영체제
