## Windows 기본 명령어

1. **DIR**: 현재 디렉터리의 파일 목록을 표시
   - 옵션: P/W/O/S/A
2. COPY: 파일을 복사
3. DEL: 파일을 삭제
4. TYPE: 파일의 내용을 표시
5. REN: 파일의 이름을 변경
6. MD: 디렉터리를 생성함
7. CD: 동일한 드라이브에서 디렉터리 위치를 변경
8. CLS: 화면의 내용을 지움
9. **ATTRIB**: 파일의 속성을 변경
    - 종류: R A S H (+ 설정 / - 해제)
10. FIND: 파일에서 문자열을 찾음
11. **CHKDSK**: 디스크 상태를 점검함
12. **FORMAT**: 디스크 표면을 트랙과 섹터로 나누어 초기화함
13. MOVE: 파일을 이동함
