## Windows 기본 명령어

1. **DIR**: 현재 디렉터리의 파일 목록을 표시
   - 옵션: P/W/O/S/A
2. COPY: 파일을 복사
3. DEL: 파일을 삭제
4. TYPE: 파일의 내용을 표시
5. REN: 파일의 이름을 변경
6. MD: 디렉터리를 생성함
7. CD: 동일한 드라이브에서 디렉터리 위치를 변경
8. CLS: 화면의 내용을 지움
9. **ATTRIB**: 파일의 속성을 변경
    - 종류: R A S H (+ 설정 / - 해제)
10. FIND: 파일에서 문자열을 찾음
11. **CHKDSK**: 디스크 상태를 점검함
12. **FORMAT**: 디스크 표면을 트랙과 섹터로 나누어 초기화함
13. MOVE: 파일을 이동함

## UNIX/LINUX 기본 명령어

1. cat: 파일 내용을 화면에 표시함
2. cd: 디렉터리의 위치를 변경
3. chmod: 파일의 보호모드를 설정하여 파일의 사용 허가를 지정
   - 사용자
     - u: user(소유자)
     - g: group(그룹)
     - o: other(다른 사용자)
     - a: all(모두)
   - 설정 기호
     - +: 권한 추가
     - -: 권한 삭제
     - =: 권한 부여
   - 권한
     - r: read(읽기)
     - w: write(쓰기)
     - x: excute(실행)
4. chown: 파일 소유자와 그룹을 변경
5. cp: 파일을 복사
6. rm: 파일을 삭제
7. find: 파일을 찾음
8. fsck: 파일 시스템을 검사하고 보수함
9. kill: PID를 이용하여 프로세스를 종료함
10. fork: 새로운 프로세스를 생성
11. killall: 프로세스의 이름을 이용하여 프로세스를 종료
12. ls: 현재 디렉터리의 파일 목록을 표시
13. mkdir: 디렉터리를 생성
14. rmdir: 디렉터리를 삭제
15. mv: 파일을 이동
16. ps: 현재 실행중인 프로세스를 표시
17. pwd: 현재 작업중인 디렉터리 경로를 화면에 표시
18. top: 시스템의 프로세스와 메모리 사용 현황을 표시
19. who: 현재 시스템에 접속해 있는 사용자를 표시
