## 매개변수 (Parameter)

#### 개념 요약

메서드 호출 시 전달하는 입력 값을 받기 위한 변수

```java
public void greet(String name) {
    System.out.println("Hello" + name);

}
```

- 위의 `name`이 매개변수 
- 호출 예: `greet("Alice")`


<br>

#### 매개변수 여러개 전달

```java
public int add(int a, int b) {
    return a + b;
}
```

- 호출 예: `add(3, 5)` ➡️ `3`과 `5`가 각각 `a`와 `b`에 전달
