## 매개변수 (Parameter)

#### 개념 요약

메서드 호출 시 전달하는 입력 값을 받기 위한 변수

```java
public void greet(String name) {
    System.out.println("Hello" + name);

}
```

- 위의 `name`이 매개변수 
- 호출 예: `greet("Alice")`
