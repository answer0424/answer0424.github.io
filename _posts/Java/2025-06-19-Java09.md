## 1차원 배열 (One-Dimensional Array)

### 개념

- 단일 행 또는 단일 행 형태로 데이터를 저장
- 배열 요소는 하나의 인덱스를 사용해 접근

<br>

#### 선언과 초기화

```java
//선언만
int[] scores;

// 생성 (크기 5)
scores = new int[5];

// 선언 + 생성 + 초기화
int[] scores = {10, 20, 30, 40, 50};
