## 필드 (Field)

#### 정의

- 객체의 속성(데이터)를 저장하는 변수
- 클래스 안에 정의되며, 객체가 생성될 때 메모리에 함께 생성

<br>

#### 예시

```java
public Class Person {
    String name;
    int age;
}
```

- 위의 `name`, `age`는 `Person` 클래스의 필드
- `Person` 객체가 만들어질 때 각각의 객체마다 따로 존재
