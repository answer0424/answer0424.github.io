## Stream API 란?

> 데이터의 흐름을 선언형으로 처리할 수 있도록 제공되는 Java API

전통적인 fop문/if문 반복 로직 대신, 가독성 높고 함수형 스타일로 데이터를 처리

```java
List<String> list = List.of("A", "B", "C");

list.stream()
    .filter(s -> !s.equals("B"))
    .forEach(System.out::println);
```
