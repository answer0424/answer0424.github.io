1. date [옵션]
- -u : UTC 국제 표준시
```bash
date
date -u
```

    <br>
2. cal [옵션] [설정값]
- -3 : 이번 달을 기준으로 전, 후 월을 보여줌
- -y : 이번 년도의 모든 월을 보여줌
- -yj : 현재 년도의 모든 월을 보여주고 총 일자 단위로 보여줌
- 월 년도 : 해당 년도의 월을 보여줌
```bash
cal
cal -3
cal -y
cal -y 2023
cal -yj
cal 05 2023
```
