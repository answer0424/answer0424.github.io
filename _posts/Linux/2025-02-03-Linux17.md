1. 디스크 볼륨 확장 순서
- PV 생성
```bash
pvcreate /dev/nvme0n3p1
```
- VG에 PV 추가
```bash
vgextend myvg /dev/nvme0n3p1
```
- LV 확장
```bash
lvextend -l +5119 /dev/myvg/mylv
```

    <br>
2. PV로 생성된 상세 확인
```bash
pvdisplay /dev/nvme0n4p1

PV Name               물리적 볼륨 명
VG Name               PV가 속한 그룹 명 
PV Size               PV 크기
Allocatable           PV 할당 가능 여부
PE Size               물리적인 PE 크기
Total PE              PE 전체 Extend 수
Free PE               사용 가능한 Extend 수
Allocated PE          할당된 Extend 수
PV UUID               PV 식별자
```
