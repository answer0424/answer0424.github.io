## wordpress 설치

```shell
# 설치
wget https://ko.wordpress.org/latest-ko_KR.zip

# 압축 해제
unzip latest-ko_KR.zip

# 파일 복사
cp -rf wordpress/ /var/www/html
```

#### 복사된 파일 수정

```shell
cd wordpress
cp wp-config-sample wp-config.php
vi wp-config.php

 23 define( 'DB_NAME', 'DB 사용자 이름' );
```
