## 산술 연산자 (Arithmetic Operators)

| 연산자 | 설명 | 예시 | 결과 |
|-|-|-|-|
| `+`  | 덧셈       | `3 + 2`  | `5`   |
| `-`  | 뺄셈       | `3 - 2`  | `1`   |
| `*`  | 곱셈       | `3 * 2`  | `6`   |
| `/`  | 나눗셈 (실수) | `3 / 2`  | `1.5` |
| `//` | 정수 나눗셈   | `3 // 2` | `1`   |
| `%`  | 나머지      | `3 % 2`  | `1`   |
| `**` | 거듭제곱     | `2 ** 3` | `8`   |
