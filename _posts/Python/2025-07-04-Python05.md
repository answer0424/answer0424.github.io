## `for` 반복문

> 파이썬의 `for`는 시퀀스(반복 가능한 객체)를 순회하는 방식

(자바의 `for-each`에 가깝고 C/Java 스타일의 `i++` 반복은 `range()`로 대체)

#### 기본 구조

```python
for 변수 in 시퀀스:
    실행문
```

<br>

#### 예제

```python
fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print(fruit)
```

<br>

#### 숫자 반복: `range()`

```python
for i in range(5):
    print(i) # 0~4
```

```python
for i in range(1, 10, 2):
    print(i): # 1, 3, 5, 7, 9
```
