## `input()` - 사용자 입력 받기

#### 기본 사용법

```python
name = input("이름을 입력하세요: ")
print("안녕하세요", name)
```

- `input()` 함수는 문자열(str)을 반환
- 사용자가 입력한 값을 `Enter`로 종료할 때까지 대기

<br>

#### 숫자 입력 주의

- 숫자를 입력받아도 문자열로 저장 (자바와 달리 자동 형변환 없음)
  
```python
age = int(input("나이 입력: " ))
print(age + 1)
```

<br>

## 자바와 비교

| 자바                                     | 파이썬               |
| -------------------------------------- | ----------------- |
| `Scanner sc = new Scanner(System.in);` | `input()`         |
| `int age = sc.nextInt();`              | `int(input(...))` |
