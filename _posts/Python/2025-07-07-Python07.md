## 함수 정의 (`def`)

#### 기본 문법

```python
def 함수이름(매개변수1, 매개변수2, ...):
    실행할 코드
    return 결과값
```

<br>

#### 예시

```python
def greet(name):
    return f"안녕하세요, {name}님!"
```

- `def` 키워드로 함수 정의 시작
- `:` 필수 (파이썬 문법)
