## 산술 연산자 (Arithmetic Operators)

| 연산자  | 의미     | 예시       | 결과                    |
| ---- | ------ | -------- | --------------------- |
| `+`  | 덧셈     | `3 + 2`  | `5`                   |
| `-`  | 뺄셈     | `3 - 2`  | `1`                   |
| `*`  | 곱셈     | `3 * 2`  | `6`                   |
| `/`  | 나눗셈    | `3 / 2`  | `1.5`  (항상 **float**) |
| `//` | 정수 나눗셈 | `3 // 2` | `1`                   |
| `%`  | 나머지    | `3 % 2`  | `1`                   |
| `**` | 거듭제곱   | `2 ** 3` | `8`                   |

<br>

#### 자바와 비교

| 자바 | 파이썬 |
| -|-|
| `/`는 정수끼리 나누면 정수 | `/`는 항상 실수 결과 |
| `Math.pow(a,b)` | `a ** b` |
