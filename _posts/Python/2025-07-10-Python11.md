## 기본 문법 구조

```python
if 조건식:
    실행문1
elif 다른조건식:
    실행문2
else:
    실행문3
```

- 들여쓰기로 코드 블록을 구분
- 중괄호 `{}` 없이 `:`와 들여쓰기로 제어
