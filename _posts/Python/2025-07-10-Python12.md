## `for` 반복문

#### 기본문법

```python
for 변수 in 시퀀스: # 시퀀스: 리스트, 문자열, 튜플, range등
    실행문
```

<br>

#### 예제: 리스트 순회

```python
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
```

출력

```ini
apple
banana
cherry
```

<br>

#### 숫자 반복: `range()`

- `range(start, stop, step)`
- `stop`은 포함하지 않음

```python
for i in range(5):
    print(i)  # 0 ~ 4
```

```python
for i in range(1, 10, 2):
    print(i)    # 1, 3, 5, 7, 9
```
