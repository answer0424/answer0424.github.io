## `split()` - 문자열 ➡️ 리스트로 나누기

#### 기본 문법

```python 
문자열.split(구분자)
```

- 기본 구분자ㅣ 공백 `' '`
- 결과는 리스트

<br>

#### 예제

```python
text = "Python is fun"
words = text.split()             # ['Python', 'is', 'fun']
csv = "a,b,c".split(",")         # ['a', 'b', 'c']
```

<br>

## `join()` - 리스트 ➡️ 문자열로 합치기

#### 기본 문법

```python
구분자.join(리스트)
```

- 리스트의 모든 요소를 문자열로 연결

<br>

#### 예제

```python
words = ["Python", "is", "fun"]
sentence = " ".join(words)       # 'Python is fun'

chars = ['a', 'b', 'c']
joined = "-".join(chars)         # 'a-b-c'
```
