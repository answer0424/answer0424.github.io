## `for` 반복문

#### 기본문법

```python
for 변수 in 시퀀스: # 시퀀스: 리스트, 문자열, 튜플, range등
    실행문
```

<br>

#### 예제: 리스트 순회

```python
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
```

출력

```ini
apple
banana
cherry
```

<br>

#### 숫자 반복: `range()`

- `range(start, stop, step)`
- `stop`은 포함하지 않음

```python
for i in range(5):
    print(i)  # 0 ~ 4
```

```python
for i in range(1, 10, 2):
    print(i)    # 1, 3, 5, 7, 9
```

<br>

## `while` 반복문

#### 기본 문법

```python
while 조건식:
    실행문
```

<br>

#### 예제

```python
count = 0
while count < 3:
    print("hello")
    count += 1
```

- 조건이 `False`가 되면 반복 종료
- `count += 1` 안하면 무한루프 주의

<br>

## `break` - 반복문 즉시 종료

```python
for i in range(10):
    if i == 5:
        break
    print(i)
```

<br>

```ini
0
1
2
3
4
```

<br>

## `continue` - 현재 반복만 건너뛰고 다음 반복

```python
for i in range(5):
    if i == 2:
        continue
    print(i)
```

<br>

```ini
0
1
3
4
```
