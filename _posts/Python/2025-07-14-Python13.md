## 클래스(Class)란

- 객체를 만들기 위한 설계도
- 함수와 변수를 묶어서 하나의 구조로 관리 가능
- 자바의 `class`와 개념 동일

```python
class Car:
    pass
```

<br>

## 인스턴스(Instance)란

- 클래스를 기반으로 생성한 실제 객체
- 클래스는 설계도, 인스턴스는 실물

```python
my_car = Car()  # Car 클래스의 인스턴스 생성
```
