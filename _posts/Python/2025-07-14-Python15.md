## 인스턴스 메서드란

- 클래스 내부에서 정의된 함수
- 반드시 첫 번째 매개변수로 `self`를 가짐
- 해당 메서드를 호출한 인스턴스를 참조

```python
class Car:
    def drive(self):  # self는 인스턴스를 가리킴
        print("자동차가 달립니다.")

my_car = Car()
my_car.drive()  # 출력: 자동차가 달립니다.
```
