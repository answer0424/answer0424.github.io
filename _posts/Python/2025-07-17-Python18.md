## `@classmethod`

#### 메서드의 요약:

- 클래스 자체를 첫번째 인자로 받음(`cls`)
- 인스턴스가 아닌 클래스 자체를 조작할 때 사용
- 보통 팩토리 메서드로 활용

<br>

#### 주요 동작 흐름:

```python
class MyClass:
    @classmethod
    def my_class_method(cls, arg1):
        print(f"클래스 이름: {cls.__name__}, 인자: {arg1}")
```
