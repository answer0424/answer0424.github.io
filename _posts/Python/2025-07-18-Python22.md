## 파일 읽기/쓰기 요약

| 기능             | 설명                        |
| -------------- | ------------------------- |
| `open()`       | 파일을 열거나 생성                |
| `readlines()`  | 파일의 모든 줄을 리스트로 읽음         |
| `writelines()` | 리스트의 각 항목을 파일에 한 줄씩 씀     |
| `with`         | 파일을 열고 자동으로 닫아주는 컨텍스트 매니저 |
