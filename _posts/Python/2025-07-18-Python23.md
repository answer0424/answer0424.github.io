## `json` 모듈 요약

| 메서드                 | 역할                     |
| ------------------- | ---------------------- |
| `json.load(f)`      | 파일에서 JSON → 파이썬 객체 변환  |
| `json.loads(str)`   | 문자열에서 JSON → 파이썬 객체 변환 |
| `json.dump(obj, f)` | 파이썬 객체 → 파일에 JSON으로 저장 |
| `json.dumps(obj)`   | 파이썬 객체 → JSON 문자열 반환   |
