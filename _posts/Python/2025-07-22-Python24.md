## `math` 모듈 - 수학 함수

| 기능        | 예시                              |
| --------- | ------------------------------- |
| 제곱근       | `math.sqrt(9)` → `3.0`          |
| 지수        | `math.exp(1)` → `2.718...`      |
| 로그        | `math.log(8, 2)` → `3.0`        |
| 삼각함수      | `math.sin(math.pi / 2)` → `1.0` |
| 반올림 없이 버림 | `math.floor(3.9)` → `3`         |
| 반올림 없이 올림 | `math.ceil(3.1)` → `4`          |
| 절댓값       | `math.fabs(-7.2)` → `7.2`       |
| 상수        | `math.pi`, `math.e`             |
