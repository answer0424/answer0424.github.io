## 람다식 문법

```python 
lambda 매개변수1, 매개변수2, ... : 표현식
```

- `def`로 정의하지 않아도 바로 쓸 수 있음
- 표현식 결과가 자동으로 `return`됨

<br>

#### 일반 함수 vs 람다식

```python
# 일반 함수
def add(x, y):
    return x + y

# 람다식
add_lambda = lambda x, y: x + y

print(add(3, 5))         # 8
print(add_lambda(3, 5))  # 8
```
