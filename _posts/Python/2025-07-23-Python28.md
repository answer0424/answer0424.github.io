## `map(function, iterable)`

> iterable의 각 요소에 함수를 적용한 결과를 반환

<br>

#### 사용 예시

```python
nums = [1, 2, 3, 4]
squared = list(map(lambda x: x ** 2, nums))
print(squared)
```
