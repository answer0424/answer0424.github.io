## 요약

| 구분 | 설명 |
|-|-|
| `pdb` | 파이썬 표준 모듈로 제공되는 텍스트 기반 디버거 |
| IDE 디버거 | PyCharm, VSCode, 등에서 GUI로 제공되는 강력한 디버깅 기능 |

<br>

## `pdb` - 파이썬 기본 디버거

#### 기본 사용법

```python
import pdb

def test():
    a = 1
    b = 2
    pdb.set_trace() # 여기서 중단
    c = a + b
    print(c)

test()
```
