## 목표 기능

| 기능 | 설명 |
|-|-|
| 연락처 추가 | 이름, 전화번호, 이메일 저장 |
| 연락처 조회 | 전체 목록 또는 이름으로 검색 |
| 연락처 삭제 | 이름으로 삭제 |
| 연락처 수정 | 특정 항목 수정 |
| 파일 저장 | JSON 파일에 자동 저장 및 불러오기 |

<br>

## 구조 예시

```text
contact_manager/
├── contact_manager.py      # 메인 로직
├── contacts.json           # 저장된 연락처 파일
└── README.md
```
