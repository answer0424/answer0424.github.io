## 개요

- RestClient - 동기, fluent API 기반 최신 클라이언트
- WebClient - 비동기/논블로킹 + fluent API
- RestTemplate - 과거부터 사용된 동기 템플릿 스타일 클라이언트
- HTTP Service Client - 인터페이스 + 어노테이션 기반 선언형 클라이언트 방식
